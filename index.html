Projects


Here are a few projects I've worked on!


Self-Parking Robot
This was my capstone project for MTE-121 (Digital Computation). We designed, constructed, and programmed an integrated EV3 Mindstorm Robot with RobotC that uses a few different sensors to actively search for a suitable parking spot in its line of sight, and perform a parallel park.


LiftBro
LiftBro is an electron application that allows you to teach a ML model what several excerises look like by demonstration, and then have LiftBro track your workout and log time spent on different movements! This was the biggest project I've developed and a great chance to learn to about ML and CV using TensorFlow and PoseNet.


ROS Motor Speed Controller Driver
A ROS driver that publishes and subscribes to a motors status (temperature, voltage), speed, and max-speed. I built the driver with C++ and wrapped it in ROS: Noetic. I got super comfortable working in Docker and Linux environments during this project.
